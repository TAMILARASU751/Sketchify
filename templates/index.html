About the Sketchify App

Project Title: Sketchify App

The Sketchify App is a web-based tool that allows users to convert their images into stunning pencil sketches or pencil art. Built using Flask, HTML, CSS, and JavaScript, the app leverages OpenCV to apply sophisticated image processing techniques, transforming everyday photos into artistic renditions with a single click.
